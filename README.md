# hyper-apex

> Monochrome [Hyper](http://hyper.is) theme based on the [Apex Atom theme](https://github.com/apex/apex-ui).

![](screenshot.png)

## Install

1. Open Hyper preferences (`~/.hyper.js`)
2. Add `hyper-apex` to the plugins list:
```
plugins: [
  'hyper-apex'
],
```
